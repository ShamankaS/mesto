# Project: Mesto


## Intro 

Mesto is a web-page with gallery of personal travel photos.
It consists of photos and a personal user profile.


[Link on GitHub Pages](https://shamankas.github.io/mesto)

## Project

Frontend (HTML, CSS, JavaScript)
